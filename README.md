#College Event Management System

A responsive and interactive College Event Management website developed using HTML, CSS, Bootstrap 5, and JavaScript.

##  Project Overview

The College Event Management System allows students to:

- View upcoming college events
- Register for events online
- View event schedules
- Browse event galleries
- Read frequently asked questions
- Contact event coordinators

The website is fully responsive and designed using Bootstrap components with a modern glassmorphism user interface.
